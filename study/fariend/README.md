## 새뜸마을 4단지 아빠친구 스터디 "파렌드"
- 요즘 아버지들은 어렵습니다. 왜?
- 우리 자라면서 배웠던 아버지와 요즘 아이들이 원하는 아빠가 너무나 다르기 때문이 아닐런지요?
- 함께 공부하고 함께 나눌 수 있으면 좀 더 스마트한 파렌드가 될 수 있을 것 같습니다.
- 파렌드(fariend = father + friend)
